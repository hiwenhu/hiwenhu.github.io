## ‚ùì Frequently Asked Questions (FAQ)

### HiSQL FAQ {#hisql-faq}

**Q1. I cannot connect to my MySQL server. What should I check?**  
- Make sure the host, port, username and password are correct.  
- Check if your MySQL server allows remote connections.  
- If you are on mobile network, confirm that your server IP/port is publicly accessible.  

**Q2. Does HiSQL support SSL connections?**  
Yes. If your server is configured with SSL, please enable SSL in the connection settings and provide the required certificates if needed.

**Q3. I found a bug in query execution. How can I report it?**  
Please email [hiamymk@gmail.com](mailto:hiamymk@gmail.com?subject=HiSQL%20Bug%20Report) with:
- A screenshot if possible
- Steps to reproduce the bug

---

### Modis FAQ {#modis-faq}

**Q1. What Redis versions does Modis support?**  
Modis is designed to work with most modern Redis versions. If you encounter any compatibility issue, please contact support with your Redis version.

**Q2. Does Modis support JSON (ReJSON)?**  
Yes. If your Redis server has the ReJSON module enabled, Modis can work with JSON data.

**Q3. The app crashes or freezes. What should I do?**  
Please try:
1. Restarting the app  
2. Restarting your device  

If the issue continues, send a bug report email including:
- Device model
- Android version
- Modis version
- Steps to reproduce the crash
